Why is JavaScript So Popular?

Top Reasons

  1. It's Easy To Learn

    2. Okay I guess that statement is pretty relative, however, it is more clear cut than other languages.

  2. It's Easy to Edit

    3. You don't need to buy anything extra that's crazy expensive. You can take a look at your work and edit it straight from the browser. Also, the results are immediate so you can see if there is still an error and what needs fixing.

  3. It's supported by modern browsers

    4. Every available browser supports Javascript so you really can't go wrong.

  4. You can Embed It In Your Other Projects

    5. While you can build projects using just JavaScript, you can always embedd JavaScript into HTML C++ and other projects.
