HTML Cheat Sheet

Basic Tags

<!DOCTYPE html> Defines the document as an HTML
<html> </html > Creates an HTML document
<head> </head> Sets off the title and other information that isnâ€™t displayed on the web page itself
<title> </title> Creates the title of the website that shows up on the browser
<body></body> Sets off the visible portion of the document

Text Tags

<h1></h1> creates the largest headline
<h6></h6> Creates the smallest headline
<p></p> Creates a paragraph
<li></li> Lists something
<ul></ul> Used to make an unordered (not numbered) list of items. You need the list tags in between these tags to get the correct formatting.
<ol></ol> Makes an ordered(numbered) list.
<em></em> Italicizes (puts emphasis on the word)
<strong></strong> Puts emphasis on a word through bold
<b></b> bolds a word
<strike> </strike> Strikesout a word
<sup> </sup> Superscript- creates smaller text above normal text
<sub> </sub> Subscript- creates smaller below normal text

Formatting Tags

<div> </div> Division of Page Content (this is where you would add your css class or id name)
<br> </br> Creates a space in the text. Line Break

Tables

<table></table> Creates a table
<tr></tr> Sets off each row in a table
<td></td> Sets off each cell in a row
<th></th> Sets off the table header (a normal cell with bold, centered text)

Links

<a href="www.website.com">This is the website </a> This creates a link to somewhere else. In between the close of the href tag and the </a you can add an image, text, whatever to connect to the link.
<a href="URL"><img src="URL"> </a> This creates an image and links it to the URL.
<a href="mailto:EMAIL"></a> Creates an email link
<a name="NAME"> </a> Creates a specific location within the document
<a href="#NAME"> </a> Links to that location from another place inside the document. ie: "back to top"
