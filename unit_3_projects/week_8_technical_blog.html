Let's Talk about Class Methods

Back to Basics

Class Methods is a method that resides at the class level. That being said, Classes are objects in ruby so the methods that are class methods only exist in the Class. For example if you were going to have class Life and a method "name" that method would only live in the "Life" class.

Ways to Define a Method

Alright so now that we've knocked out a definition, let's see how we can use these methods in action! You can define a method one of two ways

  1. use the name of the class directly as in : Class Life def Life.name "Morgan" end end
  2. or you can use the self variable (which oints to the current object(which is our definied ruby class (because the class is a definition of method class))
  3. Class Life def self.name "Morgan" end end
